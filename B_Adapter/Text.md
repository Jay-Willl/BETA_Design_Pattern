## Adapter

### Character
1. Adapter
2. Client (Main class)
3. Target (defines the performing function)
4. Adaptee (defines the "requirements")

### Two Pattern
1. Adapter extends Target implements Adaptee
2. Adapter (containing Adaptee) extends Target

### Usage
1. 更好地使用现有的类，让现有的类更好适配新的接口
2. 使得新旧版本兼容
